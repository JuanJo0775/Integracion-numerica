Puedes usar funciones como sin(x), cos(y), exp(x), log(y), etc.
Ejemplos:
  • Para \(y' = x + y\), escribe: x + y
  • Para \(y' = x^2 - \sin(y)\), escribe: x**2 - sin(y)
  • Para \(y' = e^{-x} \cdot y\), escribe: exp(-x) * y
